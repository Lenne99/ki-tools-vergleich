# KIToolsVergleich.de – Affiliate Website

## Struktur
```
affiliate-site/
├── index.html          ← Startseite
├── css/style.css       ← Komplettes Design-System
├── js/main.js          ← Filter, FAQ, Tracking
├── pages/
│   ├── buchhaltung.html
│   ├── ki-schreiben.html
│   └── ...weitere Seiten
└── vercel.json         ← Auto-Deployment Config
```

## Auf GitHub hochladen (einmalig)

1. github.com öffnen → "New repository" → Name: `ki-tools-vergleich`
2. "Upload files" klicken
3. Alle Dateien per Drag & Drop hochladen
4. "Commit changes" klicken

## Mit Vercel verbinden (einmalig)

1. vercel.com → "Add New Project"
2. GitHub Repository auswählen
3. Deploy klicken → fertig!

Ab jetzt: Jede Änderung auf GitHub = automatisch live.
